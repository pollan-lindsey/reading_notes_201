HTML

Three examples of html lists

    Unordered Lists

  • Start with a ul tag.
  • For every item in the list, use the li tag and closing tag.
  • Finish the list with a closing ul tag.

    Ordered Lists

  • Start with a ol tag.
  • For every item in the list, use the li tag and closing tag.
  • Finish the list with a closing ol tag.

    Description Lists

  • Start with a dl tag.
  • For every item in the list, use the dt tag and closing tag.
  • When describing the items in the list, use the dt tag and closing tag.
  • Finish the list with a closing dl tag.

CSS

  • A box model is something you use to format the webpage. It dictates how tall and wide everything is, and where it goes on the page. Everything on the webpage is in the box model.
  • A box model is made up of a margin, a border, and padding, all of which surround the content of the page.
  • The two ways to display an element are block and inline
  • By choosing to display the element in a block, the programmer is giving the element a rectangular space in which to be displayed. Nothing else can go in this rectangular space. An inline element is always displayed right next to the element before it, regardless of any previously specified dimensions.
  • Some units of measurement are pixels (px), em, ex, ch, rem, and many more.

Javascript

  • A condition that must be met using a statement of some kind usually containing comparison operators.
  • You write a conditional by setting a condition and then writing the appropriate statement based on the condition you want to test.

    • Three examples of comparison operators

    • === (strict equal to)
    • > (greater than)
    • <= (less than or equal to)
  • A break keyword in javascript is br put inside tags. It is used to break out of a statement the programmer wants to end.
  • A for loop is used to run the same code for a set amount of times. A while loop is used to run the same code for an unknown amount of times.
